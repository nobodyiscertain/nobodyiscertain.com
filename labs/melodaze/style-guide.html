Design System — "Stripped Down" Direction
01 — Design Principles

Record sleeve, not SaaS

Every screen should feel like a well-designed record sleeve. Clean, intentional, nothing extra. The album art does the talking.

Let it breathe

Generous whitespace everywhere. If a screen feels crowded, remove something. Space is a feature, not waste.

Monochrome + album art

The UI is black, white, and gray. The only color comes from album artwork. This makes every record pop.

02 — Color

11 — Rules
No border-radius
Everything is square. Buttons, inputs, cards, pills. Sharp corners = intentional.
No color in the UI
Black, white, grays only. Album artwork is the only source of color on any screen.
No shadows
Use borders for depth, not box-shadows. Hairline borders (1px #eee) for separation.
One typeface
Space Grotesk everywhere. Use weight (300-700) and size for hierarchy.
Labels are uppercase
Small labels use 0.7rem, uppercase, generous letter-spacing. Always muted (#999 or #bbb).
Space > decoration
When choosing between adding a visual element or adding whitespace, choose whitespace.
